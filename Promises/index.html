Promises are cool

A pattern for writing async code.

We call new promise function. This fuction have two parameters "resolve" and "reject" are the standars names. Went you call one of those parameters the promise is resolve, eather by being resolve or rejected.

How to run code if this promise was rejected or resolve.

This methods .then will run if our promise is resolve. Example:

willGetYouADog.then(() => { console.log("YAY WE GOT A DOG!!!") })

This methods .catch will run if our promise is rejected. Example:

willGetYouADog.catch(() => { console.log("NO DOG!!!") });
